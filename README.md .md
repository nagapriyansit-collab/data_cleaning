# DataClean AI - Intelligent Data Cleaning Website

A professional, AI-powered data cleaning web application built with Python Flask and modern frontend technologies. Perfect for data scientists and analysts who need automated data quality improvements.

## 🎯 Features

- **Automated Data Analysis**: Instantly analyze your data and identify quality issues
- **Smart Data Cleaning**: 
  - Remove duplicate rows
  - Handle missing values (mean, median, drop)
  - Remove outliers using IQR method
  - Remove empty columns
  - Standardize column names
- **Detailed Reports**: Get comprehensive cleaning reports and data quality scores
- **Multiple Format Support**: CSV, Excel (.xlsx, .xls), JSON
- **Fast Processing**: Optimized algorithms for handling large datasets
- **Professional UI**: Clean, modern, and intuitive user interface
- **Secure**: Data processed locally, never stored permanently

## 📋 Requirements

- Python 3.8+
- pip (Python package manager)
